Dynamic Dev Logs with JSON

Date: 7/12/2024

Welcome to the first dev log where we talk about how we dynamically load dev logs using JSON. This is a great way to manage and display content without hardcoding everything into HTML files.

Why Use JSON for Dev Logs?

Using JSON to manage your dev logs provides several benefits:

  • Flexibility: You can easily add, update, or remove logs without altering the HTML structure.
  • Scalability: JSON files can handle a large number of entries efficiently.
  • Separation of Concerns: It keeps your data separate from your presentation layer.

Implementation

Here's a brief overview of how we implemented dynamic dev logs:

  1. Created a JSON file to store dev log entries.
  2. Used JavaScript to fetch and parse the JSON data.
  3. Dynamically generated HTML elements based on the JSON data and inserted them into the DOM.

Code Snippets

Below are some key code snippets used in our implementation:


        fetch('dev-logs/dev-log-index.json')
        .then(response => response.json())
        .then(data => {
            const devLogList = document.querySelector('ul#dev-log-list');
            const maxIndex = Math.min(5, data.devLogs.length);

            for (let index = 0; index < maxIndex; index++) {
                const log = data.devLogs[index];
                const listItem = document.createElement('li');

                const link = document.createElement('a');
                link.href = "#";
                link.onclick = () => loadPage(`./dev-logs/${log.fileName}`);
                link.textContent = log.title;

                const dateSpan = document.createElement('span');
                dateSpan.textContent = ` (${log.date})`;
                dateSpan.style.marginLeft = '10px';
                dateSpan.style.color = '#555';

                listItem.appendChild(link);
                listItem.appendChild(dateSpan);
                devLogList.appendChild(listItem);
            }
        })
        .catch(error => console.error('Error fetching dev logs:', error));
