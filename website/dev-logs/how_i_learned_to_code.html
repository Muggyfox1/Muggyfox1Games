How I learned to code

Date: 7/14/2024

The learning cycle

Generally, when I learn a new code feature or language the learning cycle goes: Tutorial -> project -> more complex project.

the cycle usually ends at 1 project, and it's usually not entirely finished, but my goal was to learn the new thing rather than complete the project.

How I use the learning cycle

  1. Find a Tutorial online, hopefully a first party Tutorial, about the code language / framework / feature I want to learn.
  2. Follow the tutorial through to the end.
  3. Build a little extra past the tutorial in some way.
  4. Start a new project from scratch and start building it, using Google, ChatGPT, and documentation as a guide for how to use the features.
  5. Continue with project until I'm satisfied learning the code language / framework / feature.
